## 版本说明

版本格式：`主版本号.次版本号.修订号`

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能新增
- **修订号**：向下兼容的问题修正

## 镜像标签

- `latest` - 最新稳定版
- `v1.0.0` - 具体版本号（推荐生产环境使用）

## 获取更新

```bash
# 拉取最新镜像
docker-compose pull

# 更新并重启服务
./update.sh
```
